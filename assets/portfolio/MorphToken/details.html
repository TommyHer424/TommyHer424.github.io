Morph Token
About

MorphToken takes one cryptocurrency and outputs one or more cryptocurrencies. It operates without user accounts, allowing to quickly exchange multiple assets. Currently, the cryptocurrencies accepted are Bitcoin (BTC), Ethereum (ETH), BitcoinCash (BCH aka BCHN), Litecoin (LTC), Dash (DASH), Monero (XMR). MorphToken charges a flat fee for the output network and a dynamic service fee for trading assets.

Implementation

Developed dapps in the Ethereum blockchain using the native Mist app. Used Mist not only to store Ether or send transactions but also to implement smart contracts. Custom module build-outs and integrations were implemented to accommodate the internal business flows. A custom library was integrated to allow the easier creation of many functions that were built for improved management. Built a system that provides greater transparency and far more efficient workflows while maintaining complete control over the internal business flows.
